# Homework 3 - Survival Analysis

This homework applies parametric Accelerated Failure Time (AFT) survival models to customer churn data.

Tasks completed:
- Compared multiple parametric survival distributions
- Selected the best model using AIC/BIC
- Built a final reduced model with significant variables
- Visualized survival curves
- Calculated Customer Lifetime Value (CLV)
- Identified valuable customer segments
- Estimated annual retention budget

Tools used:
- R
- survival
- flexsurv
- ggplot2
- dplyr
